# tinkherhack-survey-analysis
Pre–post survey analysis for Tink Her Hack 4.0 (Data Cleaning, Transformation &amp; Insights)

📌 Project Overview
This project involves analysis of pre-event and post-event survey data collected for Tink Her Hack 4.0, a statewide women-in-tech hackathon.
The objective was to evaluate participant outcomes, including confidence, collaboration, and technical skill development.

🧹 Data Cleaning & Preparation
Handled messy free-text inputs where respondents entered custom responses instead of selecting predefined options
Standardized and transformed raw data into structured formats for analysis
Created categorical indexing for Likert-scale responses (Strongly Agree → Strongly Disagree)

📈 Analysis Performed
Conducted pre–post comparison analysis using matched datasets (~750 responses)
Generated insights on:
Confidence in problem-solving
Collaboration skills
Ability to build prototypes
Prepared datasets for visualization (bar charts, distributions)

🔍 Key Insights
~27% increase in confidence in building technology prototypes
~18% increase in problem-solving confidence
Improved collaboration and engagement among participants
